## Getting Started
 - ### Development / Local
    - **create .env**
      copy `.env.example` to `.env` and configure according to your system.
    - **run server**
      ```
      yarn start
      ```
    - **webpack watch**
      ```
      yarn webpack:watch
      ```
- ### Production
  - **build**
    ```
    yarn build:prod
    ```  
  - **run server**
    ```
    dist/server/index.js
    ```
